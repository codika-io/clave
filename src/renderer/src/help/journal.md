# Daily Log

The daily log gives you an AI-generated summary of everything you worked on, organized by project.

[Open Daily Log](clave://navigate/journal)

## What It Shows

Each day's log groups your sessions by project. For every session you see:

- **Session name**: The auto-generated or custom title
- **Time range**: When the session started and ended
- **Duration**: Total active time
- **AI summary**: A headline and bullet points summarizing what Claude did

Active sessions show a pulsing status dot. Completed sessions show a checkmark.

## Stats Bar

At the top of each day's log, a stats bar shows:

- **Total duration**: Combined work time across all sessions
- **Session count**: How many sessions you ran
- **Project count**: How many projects you touched

## Browsing Past Days

Use the arrow buttons to navigate to previous days, or click **Today** to jump back to the current day.

## Data Source

Session data comes from your local Claude Code history. Summaries are generated automatically. All processing happens locally.
